# Wordle with a Twist
Recreated NYT's <a href="https://www.nytimes.com/games/wordle/index.html">Wordle</a> but added additional levels. If you guess the correct 5 letter word within 6 guesses, they advance to the next round. The following rounds involve guessing words of an increasing length within 6 guesses.

This is the format (each level you get 6 guesses):
Level 1 - 5 letter word 
Level 2 - 6 letter word
Level 3 - 7 letter word
Level 4 - 8 letter word

If you can pass level 4, you win the game!
